# Castella Engine

A C++23 UCI-compatible chess engine.

## Current Features

- Magic Bitboards
- Move Ordering (stage-based)
  - History Tables
  - MVV-LVA
  - TTPV Move 
- Basic time management
- Hand-crafted eval
  - Tapered Eval
  - Material & PST
- Lazy SMP
  - Thread Voting (not currently working)
- LMR
- Aspiration Windows
- Extensions
  - Check Extensions
- Quiescence Search
